LoL 내전 ELO 정적 웹사이트

실행 방법
1. 압축을 풉니다.
2. index.html 파일을 더블클릭해서 브라우저로 엽니다.
3. 인터넷 연결 없이도 실행됩니다.

포함 데이터
- 최신내전_시즌2.xlsx의 Summary-시즌1: 시즌1 랭킹
- 최신내전_시즌2.xlsx의 Summary: 시즌2 랭킹
- 최신내전_시즌2.xlsx의 페어 승률: 페어 승률 화면
- 최신내전_시즌2.xlsx의 경기기록: 최근 경기 기록

주의
- 이 버전은 서버/DB 없이 작동하는 정적 버전입니다.
- 경기 입력 후 저장한 내용은 해당 브라우저의 localStorage에만 저장됩니다.
- 엑셀 파일이 자동으로 수정되지는 않습니다.
